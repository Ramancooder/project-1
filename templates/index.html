How it works
STEP 01
TF-IDF Vectorization
Text is converted into numerical features — words and word-pairs are scored by frequency and rarity.
STEP 02
Naive Bayes
Applies Bayes' theorem: calculates probability of spam vs ham given the feature vector.
STEP 03
Flask API
Python backend serves predictions via a REST endpoint. Frontend calls POST /predict.
STEP 04
Signal Analysis
Extra heuristics (keywords, URLs, caps) explain the model's reasoning in plain language.
